DNA Analyser

General information

Input

The sequence can be in all majuscule letters, all minuscule letters or a mix of both. Common sequence formats are accepted such as FASTA, GenBank (containing indexes and spacing) or any personalised format as long as it only contains the letters ATGC, numbers, spacing, line breaks or hyphens.

Analysis

To launch an analysis, after inputting the sequence in the box, press the "Analyse" button below the box. If the sequence is particularly long, it might take some time. The longest functions to run are the translation prediction and, to a lesser extent, the CpG islands finder.

Results display

To mantain readability, results are collapsed into categories. To display any of the categories, click on the "show" button next to it. You can click again on the ("hide") button to collapse those results again.

DNA/translated sequences encoding

DNA encoding uses the 4 letters ATGC:

A : Adenosine
T : Thymine
G : Guanine
C : Cytosine

If "format DNA" is selected in the options, additional formatting adds line breaks every 60 nucleotides, spacing every 10 nucleotides and an index before each line.

Translated sequences encoding uses one letter notation for each amino acid :

A : Alanine
C : Cysteine
D : Aspartic Acid
E : Glutamic Acid
F : Phenylalanine
G : Glycine
H : Histidine
I : Isoleucine
K : Lysine
L : Leucine

M : Methionine
N : Asparagine
P : Proline
Q : Glutamine
R : Arginine
S : Serine
T : Threonine
V : Valine
W : Tryptophan
Y : Tyrosine

* : STOP codon

Results

Sequence input

Shows the input sequence after formatting.

Nucleotide frequencies

Shows the frequency of each nucleotide. If the option "Order histogram bars" is checked, the graph will be ordered by decreasing frequencies. Highest frequency is coloured in red, higher than 25% in dark blue and lower than 25% in light blue.

Nucleotide pair frequencies

Shows the frequency of each nucleotide pair. If the option "Order histogram bars" is checked, the graph will be ordered by decreasing frequencies. Highest frequency is coloured in red, higher than 6.25% in dark blue and lower than 6.25% in light blue.

Molecular Weight

Gives the total molecular weight of the DNA sequence in Daltons (Da) and kiloDaltons (kDa) between brackets. Individual weight for each nucleotide is as follows:

A : 330.22 Da
T : 321.21 Da
G : 346.22 Da
C : 306.19 Da

Reverse/complement sequence

Gives the reverse, the complement and the reverse-complement of the DNA sequence.

CpG islands

Gives the regions of the DNA sequence that can be identified as CpG regions. These regions are richer than expected in CpG dimers in comparison to what is expected and the quantity of C/G nucleotides. A CpG island is exactly defined as a 200 nucleotide interval that has a G/C content higher than 50% and a ratio of observed/expected CpG dimers higher than 60%. Expected CpG dimers is calculated by multiplying the number of C by the number of G then dividing by 200.
Note that directly adjacent CpG islands will not be listed separately but rather put together.

Translation prediction

This category predicts the translation of the DNA sequence by comparing it to the table of codon correspondence for each amino-acid (see above for amino-acid one letter notation):

F : TTT or TTC
L : TTA or TTG or CTT or CTC or CTA or CTG
I : ATT or ATC or ATA
M : ATG
V : GTT or GTC or GTA or GTG
S : TCT or TCC or TCA or TCG or AGT or AGC
P : CCT or CCC or CCA or CCG
T : ACT or ACC or ACA or ACG
A : GCT or GCC or GCA or GCG
Y : TAT or TAC
* : TAA or TAG or TGA
H : CAT or CAC
Q : CAA or CAG
N : AAT or AAC
K : AAA or AAG
D : GAT or GAC
E : GAA or GAG
C : TGT or TGC
W : TGG
R : CGT or CGC or CGA or CGG or AGA or AGG
G : GGT or GGC or GGA or GGG

Three translation predictions are made, starting at each reading frame position.

For each translation, the frequencies of each codon are counted. If the option "Order histogram bars" is checked, the graph will be ordered by decreasing frequencies. Highest frequency is coloured in red, higher than 4.76% in dark blue and lower than 4.76% in light blue. Click on the graph to make it bigger.

Additionaly, a list of possible proteins or peptides, encoded between a start codon (M) and a stop codon (*), are displayed. Note that if protein formatting is checked in the options, the indexes before each line do not mean the position of the codons on the entire translation prediction but rather the position within the concerned "possible" protein.

Save & Load

Save results to html file

Saves the displayed results, "as is", into an html file and downloads it to the computer. Can be viewed offline later directly on the browser.

Save results in local memory

Saves the displayed results, "as is", into the HTML5 local storage of the browser. The name of the save can be set in the textbox above the button. These results can be later retrieved (see "Load from local memory" below). Local storage has a space limit of 5MB on most browsers. When full, try erasing some results (see "Erase from local memory" below).

Load/Erase results from local memory

These two buttons allow for management of local storage stored results. "Load results from local memory" will load the results selected in the dropdown menu, on a new tab of the browser. "Erase results from local memory" will permanently delete the results selected in the dropdown menu.

Load example sequence

Loads an example sequence in the input textarea. It can then be analysed by clicking the "Analyse" button.

Load sequence from text file

If the sequence you want to analyse is stored in a text file (.txt for example), it can be loaded into the textarea with this button. The sequence can then be analysed with the "Analyse" button. Note that the text file must exclusively contain a DNA sequence.

Options

Format DNA sequence

If checked, formats the DNA sequence in "Sequence input", "Reverse sequence", "Complement sequence" and "Reverse complement sequence". This formatting adds spacing every 10 nucleotides, line breaks every 60 nucleotides and an index before each line. Otherwise, the sequences contain exclusively the four nucleotide letters (ATGC) likewise to the FASTA format.

Format protein sequence

If checked, formats the amino-acid sequence in "Translation prediction from...", and "Possible proteins/peptides". This formatting adds spacing every 10 nucleotides, line breaks every 60 nucleotides and an index before each line. Otherwise, the sequences contain exclusively the codon representing letters (see above for a complete table).

Order histogram bars

If checked, the frequencies graphs bars (for nucleotides and codons) will be displayed in decreasing order, instead of order of first appearance in the sequence.

Analysis functions

Nucleotide frequency : If checked, the frequency of nucleotides in the input sequence will be calculated.

Molecular weight : If checked, the total molecular weight of the input sequence will be calculated.

CpG islands : If checked, the eventual CpG islands present in the input sequence will be found.

Translation prediction : If checked, the translation prediction will be computed, as well as its codons frequencies and the "possible" proteins or peptides.

Reverse-complement : If checked, the reverse sequence, complement sequence and reverse complement sequence will be computed.

Select/Unselect all

These buttons select or unselect the entire list of options.

Formatting tools

Format to FASTA

Formats any DNA or protein sequence (in 1 letter notation) to FASTA format. FASTA is comprised of only the characters representing the nucleotides or amino-acids, without any spacing, hyphens, etc...

Format to indexed

Formats any DNA or protein sequence (in 1 letter notation) to an indexed format. This format breaks the sequence in groups of 10, adds line breaks every 60 characters and adds an index before each line.

Format protein to 1 letter

Formats any 3-letter notation protein sequence into 1-letter notation sequence. The corresponding code for each amino-acid is the following :

A : Ala (Alanine)
C : Cys (Cysteine)
D : Asp (Aspartic Acid)
E : Glu (Glutamic Acid)
F : Phe (Phenylalanine)
G : Gly(Glycine)
H : His (Histidine)
I : Ile (Isoleucine)
K : Lys (Lysine)
L : Leu (Leucine)

M : Met (Methionine)
N : Asn (Asparagine)
P : Pro (Proline)
Q : Gln (Glutamine)
R : Arg (Arginine)
S : Ser (Serine)
T : Thr (Threonine)
V : Val (Valine)
W : Trp (Tryptophan)
Y : Tyr (Tyrosine)

* : *** (STOP codon)

Format protein to 3 letter

Formats any 1-letter notation protein sequence into 3-letter notation sequence. The corresponding code for each amino-acid is the same as above.

Isolate interval from sequence

Cuts any 1-letter sequence inserted in the box between the "start" element in that sequence to the "end" element. The same formats than for the main analysis are accepted.

PCR primer analysis

General information

This section lets you input a PCR primer (short DNA sequence used in PCR) and it will output its main properties.

Primer sequence

This is the sequence of the PCR primer, formatted to FASTA.

Length

The length of the PCR primer. The optimal length of a PCR primer is between 18 and 30 nucleotides. If the input primer is not within optimal length range, it will be displayed in red.

GC content

The content in G and C nucleotides of the PCR primer. The optimal GC content of a PCR primer is between 40 and 60% of G and C nucleotides. If the input primer is not within optimal GC content range, it will be displayed in red.

GC content in last 5 nucleotides

The GC content in the last 5 nucleotides of the PCR primer (3' end). It is recommended to have some G or C nucleotides at the end of a PCR primer. However, it is not recommended to have more than 3 G or C nucleotides. If the input primer has more than 3 G or C nucleotides in the last 5, it will be displayed in red.

Molecular weight

The molecular weight of the PCR primer, in Da.

Melting temperature (basic)

The melting temperature (Tm) of a PCR primer is the temperature at which it is expected that 50% of the primers will be dissociated from the complementary sequence. The melting temperature is calculated here using the basic approximation. The optimal melting temperature in PCR is between 52°C and 60°C. If the input primer is not within optimal Tm range, it will be displayed in red.

Higher than 4 times in a row single repeats

Informs if the sequence has a repetition of more than 4 same nucleotides in a row. It is not recommended to have longer repetitions. If the primer contains such repetitions, it will be displayed in red.

Higher than 4 times in a row pair repeats

Informs if the sequence has a repetition of more than 4 pairs of nucleotides in a row. It is not recommended to have longer repetitions. If the primer contains such repetitions, it will be displayed in red.
